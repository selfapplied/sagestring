## üéÆ System Controls

### Sobel System
- **Phase Injection**: Add fixed/osc28/osc37 phases to regions
- **Ballast Injection**: Introduce stability mass for system weighting
- **Parameter Tuning**: Adjust grid size, stability thresholds, Sobel sensitivity
- **Rendering Modes**: Wireframe, filled, or gradient visualization

### Kernel System
- **Kernel Creation**: Generate spiral, wave, or circle pattern templates
- **Evolution**: Apply FEG operators for pattern self-reflection
- **Vectorization**: Convert evolved lattices back to SVG geometry
