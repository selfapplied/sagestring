# SageString Demos

Demos showcasing the SageString library.

## Structure

- `livecam/` - Live camera edge detection demo
- `sobel/` - Sobel edge detection demos
- `svg_kernel/` - SVG kernel system demos
- `chatbot/` - Chatbot demos

## Running Demos

Open any HTML file in a web browser. Make sure to:
1. Serve from a web server (not file://)
2. Grant camera permissions for livecam demos

## Livecam Demo

The main livecam demo (`livecam/liveCam.html`) demonstrates:
- Real-time Sobel edge detection from camera
- θ⋆-parameterized fractal Sobel wavelet
- Phase-aware gain scheduling
- Continuity-regularized edge tracking
- SVG path generation
